# API Playground 🚀

A full-stack API testing playground built with **Python, Flask, SQLite, HTML, CSS, and JavaScript**.

This project allows users to test REST API requests directly from the browser — similar to a lightweight Postman.

---

## 🔥 Features

- GET, POST, PUT, PATCH, DELETE requests
- Custom endpoint input
- JSON request body support
- Response viewer
- Response headers display
- Response time measurement
- SQLite database integration
- Clear all records button
- Clickable available endpoints
- Clean UI

---

## 🛠 Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Gunicorn (deployment)
